Banco de Dados

Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas,
normalmente armazenadas eletronicamente em um sistema de computador. Um banco de dados é
geralmente controlado por um sistema de gerenciamento de banco de dados (DBMS).

Bluetooth

Bluetooth é uma tecnologia de conexão sem fio de curto alcance. Ela permite que fones de ouvido,
caixas de som, mouses e outros tipos de dispositivos sejam conectados a smartphones, tablets,
computadores, TVs e até painéis de carros.
